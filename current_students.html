MyPortal

MyPortal is an external web application used within the School of Computing in order for Students to easily access many other web based applications within and outside of the University campus. These include access to your Microsoft Outlook Web Account, access to your library account and external access to all your personal documents and other files stored on the University computer network.

Blackboard

Blackboard is a external web application used within the Scool of Computing in order for lecturers to be able to upload Lecture slides, tutorial documents and assignment breifs for students to be able to access at anytime from any computer within or outside of the University campus.

Timetables

Access each School of Computing degree timetables as well as view each Room or Facility timetables at the Stafford University campus.
